Запусти аудит: $ARGUMENTS

Шаги:
1. Определи тип аудита:
   - Код → используй refs/AUDIT_CHECKLIST.md, секция для кода (если есть refs/AUDIT_PROMPT.md — используй его)
   - Документы/процессы/стратегия → используй refs/AUDIT_CHECKLIST.md
2. Прочитай ВСЕ файлы по теме ПЕРЕД выводами
3. Проведи аудит строго по чек-листу
4. Выдай отчёт в структурированном виде

Если аргумент не указан — спроси что аудировать.
