Сохрани изменения.

Шаги:
1. git status — покажи что изменилось
2. git diff — покажи суть изменений (кратко)
3. git add — конкретные файлы (НЕ git add -A, НЕ git add .)
4. git commit — осмысленное сообщение на русском
5. git push (если remote настроен)

Не спрашивай "сохранить?" — сохраняй сразу.
Если нечего сохранять — скажи об этом.
