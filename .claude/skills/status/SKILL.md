description: Статус проекта на одном экране
---

Покажи текущий статус проекта — всё на одном экране.

Проверь и покажи:
1. Git: текущая ветка, есть ли несохранённые изменения
2. Последние 3 коммита (кратко)
3. HANDOFF.md: дата последней сессии, следующий шаг
4. ERRORS.md: есть ли нерешённые ошибки
5. reports/: сколько отчётов, последний по дате

Формат: компактная таблица или список. Без лишних слов.
