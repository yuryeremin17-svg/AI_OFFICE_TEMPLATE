Отладка проблемы: $ARGUMENTS

Если аргумент не указан — спроси что сломалось.

## Алгоритм (выполняй по порядку, не пропускай шаги)

### 1. Проверь известные ошибки
- Прочитай ERRORS.md
- Если проблема уже описана — примени известное решение и перейди к шагу 4

### 2. Найди причину
- Прочитай файл где ошибка (Read, не по памяти)
- Найди конкретную строку
- Объясни причину одним предложением

### 3. Исправь
- Покажи что меняешь и зачем
- Сделай минимальное исправление (не рефактори всё вокруг)
- Проверь что работает (запусти, протестируй)

### 4. Запиши
- Если ошибка новая — добавь в ERRORS.md (формат: проблема → причина → решение)
- Если ошибка была в ERRORS.md но решение устарело — обнови

### 5. Отчёт
```
Проблема: [что сломалось]
Причина: [файл:строка — почему]
Решение: [что сделал]
ERRORS.md: [добавлено / обновлено / уже было]
```
