# /content — Генерация контента

## Назначение
Получает идею, тезисы или материал → генерирует готовые посты для платформ клиента, адаптированные под формат и аудиторию каждой.

## Когда запускать
- Клиент говорит: "напиши пост", "контент", "content", "пост про..."
- В цепочке после research (исследование → пост)

## Параметры
| Параметр | Обязательный | По умолчанию |
|----------|-------------|-------------|
| idea (тема/тезисы) | да | - |
| platforms | нет | все из COMPANY.md |
| lang | нет | из PROFILE.md |
| style | нет | story |

## Входные данные
| Источник | Зачем |
|----------|-------|
| Идея от клиента | Сырая тема, тезисы, материал для переосмысления |
| COMPANY.md | Бренд, тон, аудитория, платформы, позиционирование |
| PROFILE.md | Голос автора, бэкграунд, стиль |

## КРИТИЧЕСКОЕ ПРАВИЛО: Только реальные факты
- ВСЕ истории, события, цифры — ТОЛЬКО от клиента или из его файлов
- НИКОГДА не выдумывать: события, клиентов, суммы, диалоги, результаты
- Если нет конкретной истории — общее наблюдение или мнение, НЕ фантазия

## Стили контента

### story (по умолчанию)
Сторителлинг из опыта. Ситуация → инсайт → вывод.

### insight
Конкретное наблюдение или вывод. Короче, острее.

### provocation
Вызов, нестандартный взгляд. Триггер дискуссии.

### how-to
Практическое руководство. Шаги, примеры, результат.

## Формат результата

Для каждой платформы клиента:

```
ПЛАТФОРМА: [название]

[Текст поста]

МЕТА:
- Хештеги: [если платформа поддерживает]
- Визуальный бриф: [формат — карусель/фото/видео, что показать]
- Лучшее время публикации: [рекомендация]
```

В конце — мета-блок:
```
ОБЩЕЕ:
- Какой пост самый сильный
- Идеи для серии (если тема тянет на 2-3 поста)
- Stories-анонс: [одна фраза + тип стикера]
```

## Правила тона
- Язык и тон из COMPANY.md и PROFILE.md
- Конкретика: цифры, примеры, ситуации
- Честно и прямо, без натужного позитива
- Не поучать — делиться опытом
- Контраст работает: "раньше/сейчас", "все думают X — а на самом деле Y"

### Запретные слова (универсальные)
- "синергия", "парадигма"
- "уникальный", "инновационный"
- "в наше время...", "мир меняется..."
- "секрет успеха"

## Верификация
- **Этап 1 — Самопроверка:**
  - Hook цепляет? Остановишься скроллить?
  - Есть конкретика или общие слова?
  - Тон соответствует бренду клиента?
  - Нет штампов и AI-звучания?
  - Длина в рамках платформы?

- **Этап 2 — Экспертная панель:**
  - **SMM-стратег:** зайдёт алгоритмам? hook, engagement, формат
  - **Целевая аудитория клиента:** "Это про меня? Интересно?"
  - **Копирайтер:** язык живой или шаблонный?

## Результат
Сохранить в `reports/content/YYYY-MM-DD-тема.md`

## Цепочки
- research → content (исследование → пост)
- content → qa (пост → проверка качества)
