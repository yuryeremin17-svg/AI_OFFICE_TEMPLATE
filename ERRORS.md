## Ошибки

### 1. Аудит без чек-листа после исправлений
- **Дата:** 17 марта 2026
- **Проблема:** После внесения исправлений по результатам аудита — закоммитил без повторной проверки по refs/AUDIT_CHECKLIST.md. Дважды в одной сессии проводил аудит "на глаз" вместо чек-листа.
- **Причина:** Спешка к коммиту. Правило "аудит = по чек-листу" не стало привычкой.
- **Решение:** Любой аудит — только по refs/AUDIT_CHECKLIST.md. После исправлений — повторный проход по чек-листу перед коммитом. Без исключений.
